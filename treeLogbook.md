## Catatan Penggunaan

- Format ini dapat langsung disalin ke `README.md` atau diunggah ke repository GitHub.
- Kode kategori dan sub-kategori dipertahankan sesuai file sumber.
- Struktur menggunakan gaya tree Unicode: `├──`, `│`, dan `└──`.
